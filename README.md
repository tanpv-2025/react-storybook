## Getting project

To clone project, you can run:

### `git clone https://github.com/tanpv-2025/react-storybook.git`

## Install package

Run command
### `yarn install`

## Start storybook

To start storybook serve, run command below
### `yarn storybook`

Open http://localhost:6006 to view storybook in the browser.
